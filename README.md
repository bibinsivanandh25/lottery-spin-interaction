# Lottery Spin Interaction

Frontend lottery spin interaction built with React and Tailwind CSS.

## What it does
- User-triggered spin rotates the number wheel
- Smooth deceleration before stop
- Fixed center highlight for winning number
- Controlled state flow (no animation glitches)

## Tech
- React
- Tailwind CSS
- JavaScript (ES6+)

## Run locally
npm install  
npm run dev

## Notes
This is a frontend interaction task.
